# Thread Kickoff Template v1

Paste this at thread start:

```md
Goal: <repo-level objective>
In-Scope: <current bounded scope>
Out-of-Scope: <explicit exclusions>
Next (Locked): <NEXT-YYYY-MM-DD-XX + one action>
To-Do List:
- P0:
- P1:
- P2:
Drift Check: OK (NO_DRIFT)

Unlock phrases (fixed):
- 确认解锁
- 确认切阶段

Two-step unlock required:
1) fixed phrase
2) target Next-ID
```
